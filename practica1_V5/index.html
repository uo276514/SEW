ASP.NET Core

¿Qué es?

Es un nuevo framework de código abierto para la creación de aplicaciones modernas conectadas a Internet, como aplicaciones web y APIs Web. Tiene mayor rendimiento que ASP.NET. Se ejecuta completo tanto en el .NET Framework de Windows como en multiplataforma .NET Core, sin embargo, ASP.NET versión Core 3 sólamente funciona en .NET Core dejando el soporte de .NET Framework.

¿Para qué surgió?

Se diseñó, en un principio, para proporcionar un framework de desarrollo optimizado para las aplicaciones que se implementan tanto en la nube como en servidores dedicados en las instalaciones del cliente.

El framework se reescribió por completo unificando lo que anteriormente había sido separado ASP.NET MVC y ASP.Web NET API a un modelo de programación único. Inicialmente surgido como ASP.NET vNext, el framework se iba a llamar ASP.NET 5, pero para evitar que fuese considerado como una actualización del ya existente finalmente se le dio le nombre de ASP.NET Core.
