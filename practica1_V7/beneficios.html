ASP.NET Core

Algunas ventajas de ASP.NET Core sobre ASp.NET son las siguientes:

  • Seguridad más estricta:
    • Menor intercambio de información y rencimiento mejorado, ya que está formado por paquetes NuGet, lo que permite una modularidad total.
    • De esta manera solo añadiremos los paquetes con la funcionalidad que necesitemos.
  • Una plataforma unificada para la creación de interfaz web y las APIs web.
  • Integración de los frameworks modernos del cliente y flujos de trabajo de desarrollo
  • Un sistema de configuración basado en la nube.
    • Preparado para su integración de forma sencilla en entornos en la nube.
  • Inyección de dependencias incorporada.
  • Capacidad para alojar en IIS u otros servidores web como Apache o self-host en su propio proceso.
  • Las peticiones HTTP se procesan siguiendo un flujo que puede ser modificado de forma modular.
    • Esto es para adaptarse a las necesidades de los usuarios (nosotros).
    • Nos permite a los usuarios poder controlar el procesado de las peticiones HTTP en nuestra aplicación.
  • Nuevas herramientas que simplifican el desarrollo web moderno.
  • Crea y ejecuta aplicaciones multiplataforma ASP.NET Core en Windows, Mac y Linux.
  • Es de código abierto y orientado a la comunidad.

Mayores ventajas

En resumen, ASP.NET Core supone una gran evolución en cuanto a frameworks de creación de aplicaciones modernas (conectadas a Internet).
